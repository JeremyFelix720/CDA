Fiche technique HTML

<a>

Element de type inline qui contient un lien hypertexte vers des pages Web internes ou externes, des fichiers, des adresses e-mail, des emplacements dans la même page, etc.

Attributs

Nom Description Valeurs (Exemple)
class Sélecteur CSS multiple dans la page. Chaine de caractères
id Sélecteur CSS unique dans la page. Chaine de caractères
style Applique un style CSS directement à l'élément (par exemple, dans le cadre d'un test). Code CSS
title Titre descriptif de l’élément. Chaine de caractères
href URL vers laquelle pointe le lien hypertexte. Hyperliens

Exemple

Wikipédia

<a href="wikipedia.org">Wikipédia</a>

<button>

Element de type inline qui indique un bouton cliquable, utilisé pour soumettre des formulaires ou acceder à une fonctionnalité

Attributs

form formtarget
Nom Description Valeurs (Exemple)
class Sélecteur CSS multiple dans la page. Chaine de caractères
id Sélecteur CSS unique dans la page. Chaine de caractères
style Applique un style CSS directement à l'élément (par exemple, dans le cadre d'un test). Code CSS
title Titre descriptif de l’élément. Chaine de caractères
hidden Masque l'élément aux visiteurs. /
formtarget Indique l'endroit où la réponse du formulaire s'ouvrira après avoir cliqué sur le bouton. "_self", "blank", "_parent", "_top", <name>"
name Nom du bouton. Chaine de caractères

Exemple

<button type="button" style="padding:5px; background-color:red;">Bouton cliquable</button>
