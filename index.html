Fiche technique HTML

<a>

Element de type inline qui contient un lien hypertexte vers des pages Web internes ou externes, des fichiers, des adresses e-mail, des emplacements dans la même page, etc.

Attributs

Nom Description Valeurs (Exemple)
class Sélecteur multiple dans la page. Chaine de caractères
id Sélecteur unique dans la page. Chaine de caractères
style Applique un style directement à l'élément (par exemple, dans le cadre d'un test). Code CSS
title Titre descriptif de l’élément. Chaine de caractères
href URL vers laquelle pointe le lien hypertexte. Hyperliens

Exemple

Wikipédia

<a href="wikipedia.org">Wikipédia</a>
